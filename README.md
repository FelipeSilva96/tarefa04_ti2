Projeto bem simples em Java feito para testar o ambiente do Eclipse e principalmente postgre e spark e também entender melhor
como funciona a integração com o GitHub. 
Além disso, nessa tarefa 04, houve integração com a nuvem Azure e uso de IA.

A ideia não foi fazer nada complexo, mas sim configurar o Eclipse, criar o projeto, versionar com Git e subir tudo para o repositório
remoto. Foi basicamente um teste para já deixar tudo organizado para trabalhos futuros. 
O projeto foi desenvolvido no Eclipse e versionado utilizando a integração do próprio Eclipse com o GitHub
